## 前端模板引擎

#### 简介

在web前端用于数据渲染，根据不同数据源重复生成结构相同的html，节省代码量，提高可维护性

##### 几种类型

- [doT](https://github.com/olado/doT)  | [doT爱好者](http://dotjs.cn/)

- [artTemplate](https://github.com/aui/artTemplate)

- [jquery-tmpl](http://github.com/jquery/jquery-tmpl)
